## Get started

1. Install git
2. Clone repository
3. Download extensions

4. Install dependencies

   ```bash
   npm install
   ```
5. Log into firebase CLI
6. ```bash
   npx firebase login
   ```

7. Fill out .env.example, then rename it to just .env

8. Run

   ```bash
   npx expo start
   ```
