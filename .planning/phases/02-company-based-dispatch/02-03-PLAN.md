  <how-to-verify>
**Prerequisites:**
- Ensure at least one company exists in Firestore with a geohash and serviceRadiusKm
- Ensure at least two drivers are registered under that company
- Both drivers should be logged in on separate devices/simulators and toggled Online

**Test 1: DISP-01 -- Nearest Company Routing**
1. Log in as commuter on a device
2. Submit a tow request from a location within the company's service radius
3. Open Firebase Console > Firestore > requests collection
4. Verify the new request document has `matchedCompanyId` set to the correct company ID
5. Verify `status` changed from `searching` to `claimed`
6. Verify `claimedByDriverId` is set to one of the company's drivers

**Test 2: DISP-02 -- Fair Driver Assignment**
1. Note which driver received the first request (Test 1)
2. Have that driver accept the request, then complete the trip
3. Submit a second tow request
4. Verify the SECOND driver receives the claim (not the same one who just completed)
5. Check Firestore: the first driver should have a more recent `lastAssignedAt` than the second

**Test 3: DISP-03 -- Decline Re-assignment**
1. Submit a new tow request
2. On the assigned driver's device, decline the request
3. Wait up to 60 seconds for the scheduler to run
4. Verify the request is re-assigned to the other driver (check `claimedByDriverId` changed)
5. On the commuter device, verify the modal stayed on "Searching for available drivers..." throughout (no interruption)

**Test 4: No Drivers -- Terminal State**
1. Toggle both drivers Offline
2. Submit a new tow request
3. Verify the request status in Firestore changes to `no_drivers`
4. On the commuter device, verify the FindingDriverModal shows "No Drivers Available" with the warning icon and "Try Again" button
5. Tap "Try Again" -- verify the modal dismisses and returns to the map

**Test 5: Admin Jobs Tab**
1. After a driver accepts a request (from any test above), check the Admin dashboard
2. Verify the Jobs tab shows the trip (this proves companyId was propagated from request to trip)
